The Government of Canadaâ€™s metadata standards are aligned with the Dublin Core Metadata Element Set, Version 1.1 and its later expanded specification, DCMI Metadata Terms, an internationally-used controlled vocabulary maintained by the Dublin Core Metadata Initiative (DCMI).

The Dublin Core Metadata Element Set is a vocabulary of 15 broad, generic properties used to describe a wide range of resources. The key value of this controlled vocabulary is that it allows a consistent approach to identifying important attributes of information resources.

DCMI Metadata is an up-to-date specification of all metadata terms including properties, vocabulary encoding schemes, syntax encoding schemes, and classes. Several of these sections specify the use of additional controlled vocabularies and classification schemes to ensure consistent metadata.

  1. title
  2. creator
  3. subject
  4. description
  5. publisher

  2. date
  3. type

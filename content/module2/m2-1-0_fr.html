Qu’est-ce que les métadonnées?

Les métadonnées sont des renseignements structurés qui décrivent, expliquent, situent ou facilitent la récupération, l’utilisation ou la gestion d’une ressource documentaire. En d’autres termes, les métadonnées sont de l’information sur de l’information.

On trouve des métadonnées partout! Elles sont dans les ressources humaines et les systèmes financiers de votre ministère, dans le catalogue d’une bibliothèque, à votre épicerie locale, à la librairie du coin et sur Internet.

  • Qui (une personne)
  • Quoi (une description de la ressource)
  • Où (un lieu)
  • Quand (un événement)

Mieux comprendre

Voici quelques exemples de métadonnées couramment enregistrées :

  • Personne : le nom de l’auteur, les participants au projet, le nom de l’utilisateur, l’expéditeur, la personne-ressource, etc
  • Description d’une ressource : le titre (d’une ressource, d’un fichier, d’un dossier, etc.), le type de ressource (une note d’information, une procédure, un ordre du jour, etc.), le sujet et les mots-clés (les mots qui décrivent un contenu), l’identificateur unique (le numéro de projet ou de contrat, le numéro d’employé, etc.), le format (Word, Excel, PowerPoint, AutoCAD, etc.)
  • Lieu : le lieu où la ressource documentaire a été publiée, où la recherche a été faite, où le siège social est situé, les coordonnées du Système d’information géographique (SIG), etc
  • Événement : la date à laquelle la ressource documentaire a été créée, reçue ou modifiée, la période couverte (exercice financier, année civile, la décennie 1990, etc.)

Using This Course

This course is a prerequisite to the classroom course Information Management Methods and Tools (I240).

It is important that you use all the course features to take full advantage of this learning opportunity, including:

  • The questions, which are designed as a learning tool;
  • The “For a Better Understanding” sections, which provide more details on specific topics;
  • The “A Word from an Expert” sections, which provide comments and suggestions from IM specialists; and
  • Activities in the “Prepare for the Classroom” sections should you plan to attend Course I240 - Information Management Methods and Tools.

You can also access the Glossary and Resources sections via the Toolbox icon at the top of each page.

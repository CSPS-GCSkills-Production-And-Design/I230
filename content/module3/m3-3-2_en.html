Function Classification Levels

Function-based classification can involve three levels of classification, which are degrees of specificity incorporated into a classification tool.

Select each level to learn more.

Function

  • Function is a broad, encompassing category. Function provides the business context to the information resource. Functions are usually standard across a department.

Sub-function

  • Sub-function is a narrower subset of the function and an optional level. Sub-functions are parts or steps in the business process used to fulfill the function. Not every function is supported by sub-functions.

Activity

  • Activity is a narrower and more specific subset of the function. Activity represents actions or tasks being performed in support of the function. Activities are usually standard across a department.
